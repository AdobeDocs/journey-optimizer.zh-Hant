# 追蹤、監視和疑難排解 {#section-overview}

全方位追蹤是有效管理客戶歷程的開端，可以主動監控、提供深入分析報告，同時有效進行疑難排解。 Adobe Journey Optimizer 提供完整的框架，可擷取客戶互動和系統事件、分析績效量度，並在發生問題時，快速解決問題。

**追蹤**&#x200B;橫跨所有管道：電子郵件、網頁、推播、應用程式內、簡訊等客戶互動，擷取開啟、點擊、轉換和自訂事件，可顯示客戶和內容互動的方式。

使用即時報告、系統警示和傳遞能力量度，協助您識別趨勢、衡量成功，及早發現潛在問題，**監控**&#x200B;歷程和行銷活動績效。

使用詳細錯誤碼、逐步執行記錄檔，解決歷程失敗、執行方式前後不一致，或是動作層面問題等目標指引，有效處理&#x200B;**疑難排解**&#x200B;問題。

本頁面彙集一些基本資源，可幫您監視歷程的健康情況，存取關鍵報告深入分析，同時有效解決客戶歷程各階段問題。

## 開始使用追蹤功能

在深入研究監視、疑難排解之前，請確保您在追蹤方面擁有扎實的基礎。 Journey Optimizer 提供全方位的追蹤功能，可擷取所有管道的客戶互動、傳遞效能及系統健康度。

瞭解如何針對電子郵件、網頁體驗、推播通知、應用程式內訊息等內容實施追蹤，包括如何設定自訂量度、設定 URL 參數，再透過內建報告，存取追蹤資料。 [開始使用追蹤](../using/start/get-started-tracking.md)

## 報告與監視

存取關鍵報告以監視歷程績效、分析量度並識別問題：

* [歷程全域報告](../using/reports/journey-global-report-cja.md) - 查看完整歷程績效指標和各大頻道上的分析。

* [歷程即時報告](../using/reports/journey-live-report.md) - 監視即時歷程執行資料，同時追蹤活躍客戶的互動。

* [歷程步驟事件](../using/reports/journey-step-events-overview.md) - 分析詳細的逐步歷程執行事件，以便進行疑難排解和最佳化。

* [開始使用報告](../using/reports/report-gs-cja.md) - 了解報告功能、量度，還有如何存取歷程洞察。

* [傳遞能力監視](../using/reports/deliverability.md) - 追蹤訊息傳遞率，還可找到跨管道的潛在傳遞問題。
