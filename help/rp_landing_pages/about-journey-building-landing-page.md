# 活動{#section-overview}

Adobe Journey Optimizer 中的活動是建構區塊，可讓您跨管道觸發事件、做出決定及採取有意義的動作，實現客戶歷程。無論是聆聽購買事件、傳送個人化電子郵件，還是即時更新輪廓，活動皆可讓您自訂歷程的每個步驟。每種類型的活動 (例如事件、協調或動作) 都扮演著獨特的角色，共同合作建立順暢、吸引人的體驗。探索這些工具時，您將了解如何使用觸發程序、條件和動作來製作如客戶需求般動態的歷程。
