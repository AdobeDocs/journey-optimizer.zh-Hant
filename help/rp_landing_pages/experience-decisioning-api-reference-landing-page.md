# API 參考{#section-overview}

Adobe Journey Optimizer 的程式碼型體驗管道可讓您在數位接觸點上，以彈性且精確的方式提供個人化產品建議。無論您是使用用戶端工具 (如 AEP Web 或 Mobile SDK)、透過 AEP Edge Network API 進行伺服器端整合，還是混合式方法，此功能都能根據您的需求適度調整。從即時互動到可擴充的後端處理，每種實施方法都有獨特的優點。在接下來的章節中，您將會探索體驗決策的基礎，了解如何針對使用案例選擇和實施正確的方法，並存取資源以簡化您的歷程。讓我們為量身打造的客戶體驗，發掘全新可能。
