# 開始使用查詢 {#queries-gs}

查詢編輯器是Adobe Experience Platform Query Service提供的互動式工具，可讓您撰寫、驗證和執行內客戶體驗資料的查詢 [!DNL Adobe Journey Optimizer] 使用者介面。

查詢編輯器支援開發查詢以進行分析和探索資料，並可讓您執行互動式查詢以用於開發用途，以及非互動式查詢以填入 [資料集](get-started-datasets.md).
