# 結束活動{#section_vqp_4ft_dgb}

**[!UICONTROL End]**&#x200B;活動可讓您標籤歷程的每個路徑的結尾。 這不是強制性的，但建議使用，以便於視覺清晰度。 事實上，如果歷程有數個結束活動，建議您在每個結尾新增標籤，讓報表更容易閱讀。 請參閱[本頁](../reports/live-report.md)。
