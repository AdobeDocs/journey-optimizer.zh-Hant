# 模擬您的歷程{#simulation}

>[!CONTEXTUALHELP]
>id="ajo_journey_simulation"
>title="模擬您的歷程"
>abstract="旅程模擬可讓您驗證您的旅程，並在啟動旅程之前查看執行績效。此模擬會使用經過訓練的模型資料在整個旅程中提供數據，以便了解該旅程在真實狀況下的表現績效。"

旅程模擬可讓您驗證您的旅程，並在啟動旅程之前查看執行績效。此模擬會使用經過訓練的模型資料在整個旅程中提供數據，以便了解該旅程在真實狀況下的表現績效。
