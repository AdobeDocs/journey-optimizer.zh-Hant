# round {#round}

傳回引數的最接近整數值，並系結捨入至正無窮大。

## 類別

數學

## 函式語法

`round(<parameters>)`

## 參數

* 小數
* 整數

## 簽名和返回類型

`round(<decimal>)`

`round(<integer>)`

傳回整數。

## 範例

`round(3.14)`

返回3。

`round(3.54)`

返回4。

`round(-3.14)`

返回–3。

`round(3)`

返回3。
