# 上 {#upper}

返回參數的大寫版本。

## 類別

字串

## 函式語法

`upper(<parameters>)`

## 簽名和返回的類型

`upper(<string>)`

返回字串。

## 範例

`upper("b")`

返回&quot;B&quot;。
