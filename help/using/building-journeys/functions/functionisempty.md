# 為空 {#isEmpty}

如果參數中的字串沒有字元，則返回true。

## 類別

字串

## 函式語法

`isEmpty(<parameters>)`

## 參數

* 字串

## 簽名和返回的類型

`isEmpty(<string>)`

返回布爾值。

## 範例

`isEmpty("")`

返回true。

`isEmpty("Hello World")`

返回false。
