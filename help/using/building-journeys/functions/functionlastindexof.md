# lastIndexOf {#lastIndexOf}

返回第二個參數上次出現的位置（在第一個參數中）。 如果沒有匹配項，則返回–1。

## 類別

字串

## 函式語法

`lastIndexOf(<parameters>)`

## 參數

| 參數 | 類型 |
|-----------|------------------|
| 字串 | 字串 |
| 指定值 | 字串 |

## 簽名和返回的類型

`lastIndexOf(<string>,<string>)`

返回整數。

## 範例

`lastIndexOf("Hello, "l")`

返回3。

說明：

在&quot;Hello&quot;中，最後一次出現的&quot;l&quot;位於位置3。
