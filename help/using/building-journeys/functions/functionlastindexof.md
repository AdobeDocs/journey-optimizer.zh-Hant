# lastIndexOf {#lastIndexOf}

傳回第二個引數最後一次出現的位置（在第一個引數中）。 如果沒有相符專案，則傳回–1。

## 類別

字串

## 函式語法

`lastIndexOf(<parameters>)`

## 參數

| 參數 | 類型 |
|-----------|------------------|
| 字串 | 字串 |
| 指定值 | 字串 |

## 簽章與傳回的型別

`lastIndexOf(<string>,<string>)`

傳回整數。

## 範例

`lastIndexOf("Hello", "l")`

傳回3。

說明：

在「Hello」中，「l」的最後一次出現位於位置3。
