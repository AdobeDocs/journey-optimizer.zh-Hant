# isNotEmpty {#isNotEmpty}

如果參數中的字串不為空，則傳回true。

## 類別

字串

## 函式語法

`isNotEmpty(<parameters>)`

## 參數

* 字串

## 簽名和返回類型

`isNotEmpty(<string>)`

傳回布林值。

## 範例

`isNotEmpty("")`

傳回false。

`isNotEmpty("hello")`

傳回true。
