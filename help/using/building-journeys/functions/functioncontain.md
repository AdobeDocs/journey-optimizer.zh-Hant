# contain {#contain}

檢查第二個引數字串是否包含在第一個引數字串中。

## 類別

字串

## 函式語法

`contain(<parameters>)`

## 參數

* 字串

## 簽章與傳回的型別

`contain(<string>,<string>)`

傳回布林值。

## 範例

`contain("rowing is great", "great")`

傳回true。
