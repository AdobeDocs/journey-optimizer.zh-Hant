# UU {#uuid}

## 類別

字串

## 函式語法

`uuid()`

## 參數

此函式不需要參數。

## 簽名和返回的類型

`uuid()`

返回字串。

## 範例

`uuid()`

返回「79e70b7f-8a85-400b-97a1-9f9826121553」.s
