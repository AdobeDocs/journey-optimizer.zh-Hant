# 修剪 {#trim}

刪除開始和結束空格。

## 類別

字串

## 函式語法

`trim(<parameters>)`

## 參數

| 參數 | 類型 |
|-----------|------------------|
| 字串 | 字串 |

## 簽名和返回的類型

`trim(<string>)`

返回字串。

## 範例

`trim(" Hello ")`

返回&quot;Hello&quot;。
