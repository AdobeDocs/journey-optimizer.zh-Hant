# lower {#lower}

傳回參數的小寫版本。

## 類別

字串

## 函式語法

`lower(<parameter>)`

## 參數

* 字串

## 簽名和返回類型

`lower(<string>)`

傳回字串。

## 範例

`lower("A")`
