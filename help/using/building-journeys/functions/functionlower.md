# 下 {#lower}

返回參數的小寫版本。

## 類別

字串

## 函式語法

`lower(<parameter>)`

## 參數

* 字串

## 簽名和返回的類型

`lower(<string>)`

返回字串。

## 範例

`lower("A")`

返回&quot;a&quot;。
