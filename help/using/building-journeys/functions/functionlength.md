# 長度 {#length}

傳回參數中字串運算式的字元數。

## 類別

字串

## 函式語法

`length(<parameters>)`

## 參數

* 字串

## 簽名和返回類型

`length(<string>)`

傳回整數。

## 範例

`length("Hello World")`

傳回11。
