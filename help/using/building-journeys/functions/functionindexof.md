# 索引 {#indexOf}

檢查第一個參數字串是否以特定字串結尾（第二個參數字串）。 如果找不到字串，則返回–1。

## 類別

字串

## 函式語法

`indexOf(<parameters>)`

## 參數

| 參數 | 類型 |
|-----------|------------------|
| 字串 | 字串 |
| 指定值 | 字串 |

## 簽名和返回的類型

`indexOf(<string>,<string>)`

返回整數。

## 範例

`indexOf("Hello", "l")`

返回2。

說明：

在&quot;Hello&quot;中，第一個出現的&quot;l&quot;位於位置2。
