# indexOf {#indexOf}

檢查第一個引數字串結尾是否為特定字串（第二個引數字串）。 如果找不到字串，則返回–1。

## 類別

字串

## 函式語法

`indexOf(<parameters>)`

## 參數

| 參數 | 類型 |
|-----------|------------------|
| 字串 | 字串 |
| 指定值 | 字串 |

## 簽名和返回類型

`indexOf(<string>,<string>)`

傳回整數。

## 範例

`indexOf("Hello", "l")`

傳回2。

說明：

在&quot;Hello&quot;中，第一個出現的&quot;l&quot;位於位置2。
