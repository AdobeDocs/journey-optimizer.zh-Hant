# 本檔案的最新更新

* [!UICONTROL Throttling rate]參數已在讀取區段活動中。 [閱讀全文](building-journeys/read-segment.md#configuring-segment-trigger-activity)
* 更新「設定決策」區段中的選件選項。 [閱讀全文](offers/offer-activities/configure-offer-selection.md)
