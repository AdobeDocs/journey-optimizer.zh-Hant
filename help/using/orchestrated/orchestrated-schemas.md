# 建立相關結構 {#orchestrated-schemas}

結構描述可表示並驗證資料的結構和格式。它提供了現實世界物件 (例如人員) 的抽象定義，並概述了該物件的每個實例中應包含哪些資料 (例如姓名、生日等)。
