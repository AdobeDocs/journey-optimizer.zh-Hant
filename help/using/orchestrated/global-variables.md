# 在協調的行銷活動中定義全域變數 {#define-global-variables}

**全域變數**&#x200B;是您在單一「協調流程」行銷活動中設定的名稱 — 值組，並在每次執行時重複使用，因此您可以使用共用值（例如預設頻道或測試電子郵件）來控制&#x200B;**[!UICONTROL 測試]**&#x200B;條件、規則產生器和其他畫布邏輯，而不需將相同的值貼到每個活動中。

此頁面說明如何定義全域變數。 一旦變數可用，如需如何在規則和&#x200B;**[!UICONTROL 測試]**&#x200B;條件中使用變數的詳細資訊，請參閱[在協調的行銷活動中使用變數](variables-orchestrated-campaigns.md)。

若要在「已協調的行銷活動」中新增或編輯全域變數，請執行下列步驟：

1. 開啟「協調流程」行銷活動。

1. 按一下&#x200B;**[!UICONTROL 儲存]**&#x200B;旁的&#x200B;**...**&#x200B;圖示，然後選取&#x200B;**[!UICONTROL 編輯變數]**。

   ![在行銷活動功能表中編輯變數選項](assets/variables-list.png){zoomable="yes"}

1. 按一下&#x200B;**[!UICONTROL 新增變數]**&#x200B;並設定變數的名稱和值。 若要編輯現有變數，請按一下&#x200B;**...**&#x200B;按鈕，然後選取&#x200B;**[!UICONTROL 編輯]**。

   ![新增或編輯變數的變數面板](assets/variables-edit.png){zoomable="yes"}

若要瞭解如何在規則和&#x200B;**[!UICONTROL 測試]**&#x200B;條件中定義全域變數，請參閱[在協調的行銷活動中使用變數](variables-orchestrated-campaigns.md#use)。
