# AI支援的品牌修正和編輯建議 {#brand-corrections-ai}

>[!AVAILABILITY]
>
>此功能適用於具有有效Brand Guidelines和AI Assistant權益的Adobe Journey Optimizer使用者。 如需存取詳細資訊，請聯絡您的Adobe代表。

## 概觀 {#overview}

Adobe Journey Optimizer將GenAI支援的品牌法規遵循工具擴充至所有支援的內容管道，包括簡訊、推播通知、網頁內容和電子郵件。 當品牌QA引擎偵測到內容違反品牌指導方針或未通過編輯品質標準時，系統會自動產生更正的替代方案，供您直接在編寫工作流程中檢閱和套用。

這會將品牌驗證從被動檢查清單轉變為主動、隨使用即修正的體驗。 內容作者不會識別違規並返回編輯器手動解決每個問題，而是會收到目標鎖定、由AI產生的建議，這些建議符合組織既定的品牌語調、語調和風格指南 — 所有這些都無需離開內容編輯器。

此功能是專為需要快速將內容從草稿移至啟用，同時維持大規模品牌相容性的內容行銷人員、撰稿人和行銷活動運運算元所設計。

## 先決條件 {#prerequisites}

在使用AI支援的品牌校正之前，請確認以下事項：

- 已在Adobe Journey Optimizer中設定&#x200B;**品牌指引**。 如果沒有有效的品牌設定檔，系統無法產生品牌一致的更正建議。
- 已為您的Adobe Experience Cloud組織啟用&#x200B;**AI助理**&#x200B;功能。
- 您有適當的許可權可建立及編輯相關管道的內容（簡訊、推播、網頁或電子郵件）。
- 在Journey Optimizer內容編輯器中至少有一個內容專案可供檢閱。

>[!NOTE]
>
>品牌修正建議是使用Adobe的創作AI基礎架構所產生，並受適用於您組織的AI助理使用原則所規範。 發佈之前請務必檢閱已接受的建議。

## 運作方式 {#how-it-works}

品牌校正會直接整合至Journey Optimizer內容編輯器中的品牌QA驗證流程。 端對端程式的運作方式如下。

**步驟1 — Brand QA掃描**：當您手動從檢閱面板或工作流程規則觸發對內容執行品牌驗證檢查時，系統會根據您設定的品牌准則評估每個內容區塊。 檢查涵蓋語調、術語、禁止的語言、編輯標準和法規要求。

**步驟2 — 違規偵測與標幟**：任何不符合品牌或編輯品質准則的內容區段都會標幟為違規指標。 違規型別（例如，膚色不符、禁止使用的字詞或指引的不合規性）會與標幟的區段一起顯示，讓作者確切瞭解需要變更的內容。

**步驟3 — AI建議產生**：對於每個已標幟的區段，Journey Optimizer會使用AI助理自動產生一或多個已更正的替代專案。 建議是以您主動的品牌准則為基礎，確保建議的文字能反映貴組織專屬的正確語調、術語和編輯風格。

**步驟4 — 內嵌預覽和檢閱**：建議的更正會內嵌顯示，直接鄰近「品牌QA」側面板中的標幟內容。 您可以比較原始文字與AI產生的替代文字，而無需離開編輯器。

**步驟5 — 接受或解除**：按一下即可接受建議，將標幟的內容取代為更正的版本。 或者，您也可以關閉建議，並手動編輯內容。 接受建議會立即更新內容區塊，並將違規標示為QA面板中已解決。

**步驟6 — 重新驗證**：套用更正後，請重新執行品牌QA掃描，以在發佈或啟用歷程或行銷活動中的內容之前，確認所有違規均已解決。

## 設定 {#configure}

除了標準先決條件以外，不需要進行其他設定。 當品牌設定檔與您的內容相關聯，且已為您的組織啟用AI助理時，功能會自動在Brand QA面板中啟動。

若要開始使用AI支援的品牌校正：

1. 開啟相關管道的內容編輯器 — 簡訊、推播通知、網頁或電子郵件。
2. 在編輯器工具列中，選取&#x200B;**品牌指引**，然後從下拉式選單中選擇適用的品牌設定檔。
3. 草稿或開啟您的內容，然後從檢閱面板中選取&#x200B;**品牌QA**&#x200B;以啟動掃描。
4. 檢閱&#x200B;**品牌QA**&#x200B;側面板中標幟的違規。 對於每個已標幟的專案，AI產生的建議會自動顯示。
5. 按一下&#x200B;**套用**&#x200B;接受建議，或按一下&#x200B;**解除**&#x200B;手動處理修正。
6. 重新執行&#x200B;**品牌QA**&#x200B;以確認所有違規均已解決，然後繼續您的標準核准或啟動工作流程。

### 支援的管道 {#supported-channels}

Adobe Journey Optimizer的下列內容型別提供AI支援的品牌校正：

| 頻道 | 支援的內容元素 |
|---|---|
| **電子郵件** | 主旨列、前置標題、內文、CTA標籤 |
| **簡訊** | 訊息內文 |
| **推播通知** | 標題，內文 |
| **網頁** | 標題、正文、按鈕標籤 |

>[!NOTE]
>
>支援的內容元素可能會因您的頻道設定和品牌設定檔中定義的品牌指引而有所不同。 影像和視覺資產驗證不在AI產生文字校正的範圍中。

## 主要優點 {#key-benefits}

**減少手動編輯工作量**：內容作者不再需要手動針對每個標幟的問題互動參照品牌准則。 AI建議會顯示可立即套用的替代方案，大幅減少在修正週期所花費的時間。

**一致的品牌法規遵循**：更正是以用於驗證的相同品牌准則為基礎。 接受的建議會維持與所有管道中核准的品牌聲音和編輯標準的一致性，減少多管道行銷活動中訊息不一致的風險。

**更快速的內容製作**：藉由將品牌QA變成簡化的隨用即用工作流程，團隊可更快速地移動內容於稽核週期，縮短草稿與啟用之間的轉換時間。 Campaign運運算元只需一次即可解決整組違規，無需在工具之間切換。

**跨頻道涵蓋範圍**：不論是製作簡訊行銷活動、行動推播序列或網頁應用程式內訊息，品牌修正建議都可在所有支援的內容介面中一致地取得，以確保品牌標準在您的品牌溝通的任何地方都得到維護。
