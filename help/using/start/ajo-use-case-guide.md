# 為您的目標尋找合適的Journey Optimizer功能 {#ajo-use-case-guide}

>[!BEGINSHADEBOX]

**在此頁面上：**&#x200B;從您要完成的任務開始，然後跳至解決該問題的Adobe Journey Optimizer功能 — 不需要先知道功能名稱。

>[!ENDSHADEBOX]

[!DNL Adobe Journey Optimizer]提供許多功能，而正確的功能取決於您嘗試達成的目的。 本指南是根據業務目標而非產品功能而編排：尋找符合您需求的目標，然後按照連結開始使用建議的功能。

使用此頁面作為快速路由器 — 掃描您的目標並直接跳至正確的功能。 若您想依照引導式、逐步指示和影片進行，請改從[入門中心](onboarding-hub.md)開始。

如果特定案例沒有提供端對端教學課程，連結會帶您前往目前的最佳起點，以瞭解功能並開始進行。

AI內建於許多這些功能中 — 在下表中尋找&#x200B;**(AI)**&#x200B;標籤。 對話式[AI助理](ai-features.md#ai-assistant)也可以隨時回答產品問題，並呈現有關您歷程的營運深入分析。 如需完整的智慧型功能集，請參閱[AI與智慧型功能](ai-features.md)。

>[!TIP]
>
>不熟悉Journey Optimizer？ 從[入門中心](onboarding-hub.md)開始，瞭解逐步指示、已組織的使用案例和影片。 您也可以瀏覽完整的[Journey Optimizer教學課程](https://experienceleague.adobe.com/zh-hant/docs/journey-optimizer-learn/tutorials/overview){target="_blank"} （影片和引導式逐步說明）、觀看專家組織的[影片播放清單](https://experienceleague.adobe.com/en/playlists?solution=Journey+Optimizer){target="_blank"}，以及在[訓練沙箱](https://experienceleague.adobe.com/zh-hant/docs/journey-optimizer-learn/configure-a-training-sandbox/introduction-and-prerequisites){target="_blank"}中或在[實作挑戰](https://experienceleague.adobe.com/zh-hant/docs/journey-optimizer-learn/challenges/introduction-and-prerequisites){target="_blank"}中練習。
