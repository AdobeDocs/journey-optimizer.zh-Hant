# 此工作適用的Journey Optimizer功能為何？ {#ajo-use-case-guide}

>[!BEGINSHADEBOX]

**在此頁面上：**&#x200B;從您要完成的任務開始，然後跳至解決該問題的Adobe Journey Optimizer功能 — 不需要先知道功能名稱。

>[!ENDSHADEBOX]

[!DNL Adobe Journey Optimizer]提供許多功能，而正確的功能取決於您嘗試達成的目的。 本指南是根據業務目標而非產品功能而編排：尋找符合您需求的目標，然後按照連結開始使用建議的功能。

如果特定案例沒有提供端對端教學課程，連結會帶您前往目前的最佳起點，以瞭解功能並開始進行。

AI內建於許多這些功能中 — 在下表中尋找&#x200B;**(AI)**&#x200B;標籤。 對話式[AI助理](ai-features.md#ai-assistant)也可以隨時回答產品問題，並呈現有關您歷程的營運深入分析。 如需完整的智慧型功能集，請參閱[AI與智慧型功能](ai-features.md)。

>[!TIP]
>
>不熟悉Journey Optimizer？ 瀏覽完整的[Journey Optimizer教學課程](https://experienceleague.adobe.com/zh-hant/docs/journey-optimizer-learn/tutorials/overview){target="_blank"} （影片和引導式逐步解說）、觀看專家組織的[影片播放清單](https://experienceleague.adobe.com/zh-hant/playlists?solution=Journey+Optimizer){target="_blank"}，並在[訓練沙箱](https://experienceleague.adobe.com/zh-hant/docs/journey-optimizer-learn/configure-a-training-sandbox/introduction-and-prerequisites){target="_blank"}或是[實作挑戰](https://experienceleague.adobe.com/zh-hant/docs/journey-optimizer-learn/challenges/introduction-and-prerequisites){target="_blank"}中練習。

## 不確定？ {#not-sure}

如果您的目標對應到您不熟悉的術語，或您不確定表格指向哪個功能，請從[Journey Optimizer重要術語](terminology.md)頁面開始，以釐清每個功能背後的概念。

您也可以透過[Journey Optimizer教學課程](https://experienceleague.adobe.com/zh-hant/docs/journey-optimizer-learn/tutorials/overview){target="_blank"}中的端對端練習來建立實作信賴度。
