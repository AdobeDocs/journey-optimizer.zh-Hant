# Journey Optimizer功能可用性 {#ajo-features-availability}

>[!BEGINSHADEBOX]

**在此頁面上：**&#x200B;瞭解哪些[!DNL Adobe Journey Optimizer]功能可用、每個功能屬於哪個生命週期階段（一般可用性、有限可用性或Beta）、它們適用於哪個基本選件，以及它們何時送出 — 因此您可以回答&#x200B;*「我可以使用這個嗎？」* 而不需深入瞭解發行說明。

>[!ENDSHADEBOX]

此頁面整合[!DNL Adobe Journey Optimizer]中的功能可用性，以便您可以確認在預先實作範圍設定期間可以使用的功能。 功能會依功能區域分組。 在每個區域中，每個功能都會列出其目前的生命週期狀態、其所套用的基本選件、其可用日期，以及有關設定或區域限制的任何附註。

在&#x200B;*自*&#x200B;起可用的資料行中標示為&#x200B;**核心功能**&#x200B;的資料列是長期存在的基本功能，在2026年以前便已普遍可用。 日期資料列反映2026年出貨的變更。

>[!IMPORTANT]
>
>**為什麼在我的環境中看不到功能？** **有限可用性**&#x200B;或&#x200B;**Beta**&#x200B;中的功能未對所有人可見 — 這些功能會先推出至一組受限制的組織。 如果您所讀到的功能未出現在環境中，請檢查其狀態如下：如果它是&#x200B;**LA**&#x200B;或&#x200B;**Beta**，請聯絡您的Adobe代表以要求存取權。 此處列出的功能並不表示已在您的環境中啟用。

>[!NOTE]
>
>**可用性與權利。** 此頁面追蹤&#x200B;*功能生命週期與可用性* （功能是否已出貨及成熟度）。 授權中是否包含&#x200B;*功能*&#x200B;取決於您的基本選件和附加元件 — 請參閱[套件和功能](ajo-packages.md)。

## 生命週期狀態的意義 {#status-definitions}

| 狀態 | 其含義 |
|--------|--------------|
| **GA** — 一般可用性 | 已發佈至所有環境。 適用於授權包含權能的任何組織。 |
| **LA** — 可用性限制 | 已發行給一組受限制的組織。 **請聯絡您的Adobe代表**&#x200B;以要求存取權。 |
| **Beta** | 搶先使用評估版。 可能在一般可用性之前變更。 可能需要選擇加入。 |

## 如何「套用至」對應至基本優惠方案 {#applies-to}

**套用至**&#x200B;資料行參考到三個[!DNL Adobe Journey Optimizer]基本選件：

- **Journey Optimizer — 行銷活動** — 批次、對象型協調
- **Journey Optimizer — 歷程** — 即時、事件導向的協調流程
- **Journey Optimizer — 行銷活動和歷程** — 兩者

標示為&#x200B;**所有基本選件**&#x200B;的頻道、內容和平台功能不論基本選件為何，皆可使用，但大部分仍需要相關的頻道或進階功能附加元件。 如需權利詳細資訊，請參閱[套件和功能](ajo-packages.md)。
