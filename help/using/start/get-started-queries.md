# 開始查詢 {#queries-gs}

查詢編輯器是Adobe Experience Platform查詢服務提供的互動式工具，它允許您編寫、驗證和運行查詢，以在 [!DNL Adobe Journey Optimizer] 用戶介面。

查詢編輯器支援為分析和資料探索開發查詢，並允許您運行互動式查詢以用於開發目的以及非互動式查詢以填充 [資料集](get-started-datasets.md)。


![](../assets/queries-home.png)
