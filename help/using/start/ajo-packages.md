# 我的[!DNL Adobe Journey Optimizer]封裝中有哪些可用專案？ {#ajo-packages}

[!DNL Adobe Journey Optimizer]功能會因您的授權合約、啟用的頻道和使用者許可權而有所不同。 使用本指南瞭解套件中通常提供哪些功能，並直接導覽至每項功能的產品檔案。

是否可用也取決於通道設定、實作先決條件和購買的附加元件。 選取與您的授權模式相符的標籤。

>[!BEGINTABS]

>[!TAB 歷程與行銷活動]

此標籤適用於目前的[!DNL Adobe Journey Optimizer]模組化封裝模式（AJO歷程、AJO行銷活動或AJO行銷活動和歷程）下獲授權的客戶。

## 基礎封裝 {#current-packages}

| 套件 | 最適合 |
|---------|----------------|
| **AJO行銷活動** | 跨傳出頻道排程、批次和異動訊息。 |
| **AJO歷程** | 事件觸發、即時歷程協調和自動化。 |
| **AJO行銷活動和歷程** | 完整的平台 — 結合排程訊息和即時協調。 |

下列附加元件可在任何基本套件之上擴充頻道和功能涵蓋範圍：

| 附加元件 | 解鎖的內容 |
|--------|----------------|
| **傳出** | 電子郵件、簡訊/多媒體簡訊、推播通知和直接郵件 |
| **行動應用程式** | 應用程式內訊息和行動推播 |
| **網頁** | Web管道、程式碼型體驗和內容卡 |
| **決策** | 即時優惠決定和AI支援的最佳化 |

## 封裝摘要 {#package-summaries}

+++**選取**

最適合組織批次和異動訊息快速入門：

- 排程的批次行銷活動和交易式訊息
- 核心行銷活動和歷程執行
- 電子郵件、簡訊、推播和自訂動作頻道基礎
- 標準協調護欄

+++

+++**Prime**

包括Select中的所有功能，以及即時協調和傳入頻道：

- 即時、事件觸發的歷程協調
- 傳入頻道：網頁、應用程式內傳訊、程式碼型體驗、內容卡片和直接郵件
- 進階對象細分與目標定位

+++

+++**Ultimate**

包括Prime中的所有功能，以及決策和進階最佳化：

- 即時優惠決定和個人化
- AI支援的排名和最佳化模型
- 進階報告和實驗功能

+++
