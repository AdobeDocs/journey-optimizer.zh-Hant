# 我的[!DNL Adobe Journey Optimizer]封裝中有哪些可用專案？ {#ajo-packages}

[!DNL Adobe Journey Optimizer]功能會因您的授權合約、啟用的頻道和使用者許可權而有所不同。 使用本指南瞭解套件中通常提供哪些功能，並直接導覽至每項功能的產品檔案。

是否可用也取決於通道設定、實作先決條件和購買的附加元件。 選取與您的授權模式相符的標籤。

>[!BEGINTABS]

>[!TAB 歷程與行銷活動]

此標籤適用於目前的[!DNL Adobe Journey Optimizer]模組化封裝模式（Journey Optimizer — 行銷活動、Journey Optimizer — 歷程或Journey Optimizer — 行銷活動與歷程）下獲授權的客戶。

## 基礎封裝 {#current-packages}

| 套件 | 包含的內容 |
|---------|----------------|
| **Journey Optimizer — 行銷活動** | 行銷活動策劃：批次參與的單一或多步驟對象工作流程。 透過電子郵件、推播和簡訊傳送異動訊息。 |
| **Journey Optimizer — 歷程** | 即時Journey Orchestration：事件觸發的歷程，支援串流和批次處理。 透過電子郵件、推播和簡訊傳送異動訊息。 |
| **Journey Optimizer — 行銷活動和歷程** | Campaign Orchestration與Real-time Journey Orchestration結合。 透過電子郵件、推播和簡訊傳送異動訊息。 |

>[!NOTE]
>
>總資料量權益依封裝而不同： **行銷活動**&#x200B;客戶每個可定址設定檔有權使用15 KB；**歷程**&#x200B;和&#x200B;**行銷活動和歷程**&#x200B;客戶每個可定址設定檔有權使用75 KB。

下列附加元件可在任何基本套件之上擴充頻道涵蓋範圍。 **所有管道**&#x200B;附加元件將傳出傳遞、行動裝置和網頁組合在一起。

| 附加元件 | 管道已解鎖 |
|--------|----------------|
| **傳出傳遞** | 電子郵件、推播通知、直接郵件。 包含傳遞能力基本知識。 |
| **行動** | 行動介面的應用程式內訊息、推播通知、內容卡和程式碼型頻道 |
| **網頁** | 網頁表面的網頁頻道和程式碼型頻道 |
| **所有頻道** | 套件組合傳出傳遞+行動+網頁 |
| **決策** | 即時優惠決定和AI支援的最佳化 |

## 封裝摘要 {#package-summaries}

+++**選取**

最適合組織批次和異動訊息快速入門：

- 排程的批次行銷活動和交易式訊息
- 核心行銷活動和歷程執行
- 電子郵件、簡訊、推播和自訂動作頻道基礎
- 標準協調護欄

+++

+++**Prime**

包括Select中的所有功能，以及即時協調和傳入頻道：

- 即時、事件觸發的歷程協調
- 傳入頻道：網頁、應用程式內傳訊、程式碼型體驗、內容卡片和直接郵件
- 進階對象細分與目標定位

+++

+++**Ultimate**

包括Prime中的所有功能，以及決策和進階最佳化：

- 即時優惠決定和個人化
- AI支援的排名和最佳化模型
- 進階報告和實驗功能

+++
