# AI支援的品牌修正 {#brand-corrections}

>[!AVAILABILITY]
>
>此功能適用於Adobe Journey Optimizer，並適用於推播、簡訊和網頁內容頻道。

## 概觀 {#overview}

在品牌QA期間標籤內容時，Adobe Journey Optimizer可以使用產生式AI自動產生修正或改良的替代文字。 您不會手動重寫已標幟的副本，而是會收到符合您品牌指引的內嵌建議 — 您只需按一下即可檢閱、預覽和套用這些建議。

這會將品牌QA步驟從封鎖檢閱閘道轉換為&#x200B;**隨用即修體驗**，減少手動修正所花費的時間，並加快跨管道的內容製作。

此功能是專為需要維持高容量、多頻道行銷活動品牌合規性，同時又不減慢製作工作流程的內容行銷人員、撰稿人和行銷活動運運算元所設計。

## 運作方式 {#how-it-works}

Adobe Journey Optimizer中的品牌QA會根據您組織的品牌指導方針來評估您的內容，包括語調、術語、訊息標準和編輯規則。 偵測到違規或品質問題時，系統會標示受影響的內容元素，並在支援的情況下，使用Adobe產生AI引擎自動產生建議的替代專案。

端對端流程如下：

1. **品牌QA掃描** — 當您對內容（推播通知內文、SMS訊息文字或Web內容區塊）執行品牌檢查時，系統會根據有效品牌規則評估每個元素。
2. **違規偵測** — 一個或多個品牌條件失敗的元素會標示嚴重度指標（例如，嚴重、警告或建議）。
3. **AI建議產生** — 對於每個標幟的元素，系統會自動產生一或多個已更正的替代文字。 建議是由Adobe產生的AI引擎所產生，該引擎知道違規原因和您品牌的指南內容。
4. **內嵌預覽** — 建議的取代文字會內嵌顯示，直接與標幟的原始文字並排。 在進行任何變更之前，您可以並排比較原始版本和建議版本。
5. **一鍵套用** — 如果建議符合您的需求，請選取&#x200B;**套用**，將標幟的文字取代為AI產生的版本。 內容編輯器會立即更新，且品牌標幟會被清除。
6. **手動覆寫** — 您從未被鎖定在AI建議中。 您可以在套用建議之前編輯建議、將其關閉並撰寫自己的更正，或如果內容允許例外情況，則完全忽略標幟。

>[!NOTE]
>
>AI產生的建議僅為建議。 您的團隊保留完整的編輯控制權。 套用建議之前，請一律在行銷活動內容中檢閱建議。

## 先決條件 {#prerequisites}

在使用AI支援的品牌校正之前，請確保符合以下條件：

- **已設定品牌指引** — 貴組織必須在Adobe Journey Optimizer中設定至少一個使用中的品牌設定檔。 品牌設定檔定義評估內容所依據的規則。 請聯絡您的Adobe管理員或品牌管理員，確認品牌設定檔已發佈並與您的沙箱建立關聯。
- **已啟用Content AI功能** — 必須為您的組織和沙箱啟用AI支援的內容協助。 這是在Adobe Admin Console中的產品設定檔層級管理。 如果品牌掃描後未出現AI建議，請向您的管理員確認AI內容產生權益為使用中。
- **支援的內容型別** — 下列內容型別支援含AI建議的品牌更正：推播通知標題和內文、SMS訊息文字，以及透過Journey Optimizer Web Designer編輯的網頁內容區塊。 多媒體資產（影像、視訊）會分別評估品牌法規遵循，且不在文字層級AI校正的範圍中。
- **編輯許可權** — 您必須擁有標籤內容所在歷程、行銷活動或內容範本的編輯存取權。

## 設定 {#configure}

如果您的組織已使用Brand QA，則啟用AI支援的品牌校正不需要額外設定。 在支援的內容型別中偵測到品牌違規時，AI建議層會自動啟用。

若要執行品牌檢查並存取AI建議：

1. 在Adobe Journey Optimizer中開啟您的行銷活動或歷程。
2. 導覽至相關管道（推播、簡訊或網頁）的內容步驟。
3. 選取&#x200B;**檢查品牌一致性** （或從內容編輯器工具列開啟&#x200B;**品牌QA**&#x200B;面板）。
4. 等候掃描完成。 已標幟的元素會在畫布中反白顯示，並列在右側的&#x200B;**品牌問題**&#x200B;面板中。
5. 對於每個已標幟的專案，展開問題列以檢視違規原因和AI產生的建議。
6. 使用&#x200B;**預覽**&#x200B;檢視內容畫布中呈現的建議文字。
7. 選取&#x200B;**套用建議**&#x200B;以取代標幟的文字，或選取&#x200B;**編輯**&#x200B;以在套用之前修改建議。
8. 解決或確認所有問題後，請重新執行品牌檢查以確認法規遵循。

>[!NOTE]
>
>套用建議會更新編輯器中的即時內容，但不會自動發佈或啟動行銷活動。 您必須正常完成其餘的行銷活動設定和啟動步驟。

### 使用多個建議 {#multiple-suggestions}

對於某些違規，AI引擎可能會產生多個替代方案。 在這些情況下，**品牌問題**&#x200B;面板會顯示附編號的選項清單。 在選取最符合您意圖的箭頭之前，使用前進和後退箭頭來循環切換替代箭頭。 每個替代方案都是以不同的文體方法產生 — 例如，一個方案可能優先強調簡潔，而另一個方案則強調色調對齊。

### 大量更正 {#bulk-corrections}

如果相同內容片段中有多個元素被標幟，您可以個別套用建議，或使用&#x200B;**品牌問題**&#x200B;面板頂端的&#x200B;**套用所有建議**&#x200B;動作，在單一步驟中接受所有AI產生的取代。 在使用大量套用之前，請仔細檢閱清單，因為此動作會同時取代所有標幟的文字。

>[!NOTE]
>
>只有當所有標幟的專案都有相關聯的AI建議時，才能使用大量套用。 沒有可用建議的專案（例如影像法規遵循旗標）會被略過，並仍會標幟以供手動檢閱。
