# 輸送量的效能附加元件 — （推播）單一 — 訊息傳送 {#performance-add-on-for-throughput-push-unitary-mes}

>[!AVAILABILITY]
>
>此功能可從&#x200B;**AJO26.7** (2026-07-27)取得。

## 概觀 {#overview}

Adobe Journey Optimizer推出輸送量的&#x200B;**效能附加元件 — （推播）單一 — 訊息傳送**，讓組織能夠跨推播頻道提供更相關、個人化的客戶體驗。

本頁說明如何在您的行銷活動和歷程中設定和使用此功能。

## 先決條件 {#prerequisites}

開始之前：

* 您可使用必要的&#x200B;**推播**&#x200B;許可權來存取Adobe Journey Optimizer。
* 已設定推播通道表面。 請參閱[設定推播通道](../configuration/channel-surfaces.md)。

## 運作方式 {#how-it-works}

輸送量的效能附加元件 — （推播）單一 — 訊息傳送直接與AJO執行引擎整合。 當設定檔達到歷程或行銷活動中的推播動作時，功能會在傳送時套用已設定的引數。

主要功能：

* **設定檔層級個人化** — 使用設定檔和內容屬性調整每個收件者的設定。
* **歷程與行銷活動支援** — 適用於協調的歷程與一次性行銷活動。
* **即時量度** — 結果會出現在[推播報告](../reports/push-report.md)中。

## 設定傳輸量的效能附加元件 {#configure}

1. 在AJO左側功能表中，導覽至&#x200B;**管道** > **推播設定**。
1. 選取或建立通道設定。
1. 在&#x200B;**的**&#x200B;效能附加元件區段中，啟用此功能。
1. 設定必要的引數。
1. 按一下「**儲存**」。

>[!NOTE]
>
>設定變更會套用至新的歷程執行。 進行中的歷程不受影響。
