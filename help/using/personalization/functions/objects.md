# 物件函式 {#objects}

## 為null{#isNull}

此 `isNull` 函式確定對象引用是否不存在。

**格式**

```sql
{%= isNull(object) %}
```

**範例**

以下操作將檢查該人員的家庭地址是否不存在。

```sql
{%= isNull(person.homeAddress) %}
```

## 非空{#isNotNull}

此 `isNotNull` 函式確定對象引用是否存在。

**格式**

```sql
{%= isNotNull(object) %}
```

**範例**

以下操作將檢查該人員的家庭地址是否存在。

```sql
{%= isNotNull(person.homeAddress) %}
```
