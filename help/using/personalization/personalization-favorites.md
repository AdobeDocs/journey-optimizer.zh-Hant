# 將屬性新增至我的最愛 {#fav}

>[!BEGINSHADEBOX]

**在此頁面上：**&#x200B;瞭解如何在您的最愛功能表中新增、存取及移除屬性，以便在Adobe Journey Optimizer中建立個人化運算式時快速重複使用。

>[!ENDSHADEBOX]

將不同屬性加入收藏夾功能表可讓您快速存取最常使用的專案。 若要新增屬性至您的最愛，請按一下橢圓選單，然後選擇&#x200B;**[!UICONTROL 新增至我的最愛]**。

<!--
![](assets/favorite-option.png)
-->

若要存取您最愛的專案，請使用左窗格中的&#x200B;**[!UICONTROL 我的最愛]**&#x200B;功能表。

您可以從此清單快速將個人化物件新增至您目前的運算式。

<!--
![](assets/favorite-list.png)
-->

如果您不想再看到我的最愛清單中的專案，可以將它從我的最愛移除。

<!--
![](assets/favorite-remove.png)
-->

>[!TAB 概觀]

**TL；DR**

此頁面說明如何將常用屬性新增至個人化編輯器中的「我的最愛」功能表、存取這些屬性以快速重複使用，以及在不再需要這些屬性時將其移除。

**個意圖**

* 將個人化屬性新增至我的最愛功能表以進行快速存取
* 從編輯器左窗格的「我的最愛」功能表存取我的最愛屬性
* 從我的最愛清單移除屬性

>[!TAB 字彙]

* **我的最愛功能表**：個人化編輯器左側導覽窗格中的專屬區段，可讓您快速存取使用者標示為我的最愛屬性，以加快運算式建置速度。 *（產品特定）*

>[!TAB 術語]

* **正式名稱：**&#x200B;我的最愛 — 變體：我的最愛功能表、屬性我的最愛、我的最愛屬性
* **正式名稱：**&#x200B;我的最愛 — 變體：我的最愛功能表，我的最愛屬性

>[!TAB 常見問題集]

**問：如何將屬性新增至我的最愛？**

按一下導覽窗格中屬性旁的省略符號選單，然後選擇&#x200B;**新增至我的最愛**。

**問：我可以在哪裡找到我最喜愛的屬性？**

在個人化編輯器左窗格的&#x200B;**我的最愛**&#x200B;功能表中。

**問：如何從我的最愛移除屬性？**

從左窗格中的&#x200B;**我的最愛**&#x200B;功能表，從我的最愛移除專案。

>[!ENDTABS]
