[!DNL Adobe Journey Optimizer]持續提供新功能、現有功能的增強功能並修正錯誤。 所有變更都會在每月最後一週整合於發行說明。

**設定**

* 您現在可以在通道表面層級選取行銷動作。在表面中使用時，會運用與該行銷動作相關的所有同意政策，以尊重客戶的偏好設定。 [閱讀全文](../action/consent.md#surface-marketing-actions)
* 現在管道表面可以使用「物件層級存取控制」。 [閱讀全文](../configuration/channel-surfaces.md#create-channel-surface)
* 在管道介面中啟用清單取消訂閱時，您現在可以定義同意層級，以符合您管理所有其他來源同意的方式。 [閱讀全文](../email/email-settings.md#list-unsubscribe)

**內容管理**

* 您現在可以模擬所有管道的內容範本。 [閱讀全文](../content-management/content-templates.md#test-templates)

**個人化**

* 新的 **toInt** 協助程式函式可在運算式編輯器中使用。 它可讓您將任何這些型別（數字、雙精度、整數、長整數、浮點數、短整數、位元組、布林值、字串）轉換為整數。 [閱讀全文](../personalization/functions/math.md#to-int)
