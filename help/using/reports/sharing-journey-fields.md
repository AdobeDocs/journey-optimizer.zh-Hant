# 歷程欄位 {#sharing-journey-fields}

![](../assets/do-not-localize/badge.png)

此混合用於歷程結構（與journeyStepEvent相關）。 它包含下列欄位：

## journeyID

主要歷程的ID。

類型: 字串

## journeyVersionID

歷程版本ID。 此id代表歷程的身分。

類型: 字串

## 名稱

歷程的名稱。

類型: 字串

## 說明

歷程說明。

類型: 字串

## 版本

版本，表示為`major`。`minor`

類型: 字串
