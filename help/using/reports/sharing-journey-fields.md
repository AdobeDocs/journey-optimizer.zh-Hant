# 歷程欄位 {#sharing-journey-fields}

此欄位組用於行程架構（與journeyStepEvent相關）。 它包含以下欄位：

## 旅程ID {#journeyid-field}

主行程的ID。

類型: 字串

## journeyVersionID {#journeyversionid-field}

行程版本的ID。 此id表示旅程的標識。

類型: 字串

## 名稱 {#name-field}

旅程名稱。

類型: 字串

## 描述 {#description-field}

旅程描述。

類型: 字串

## 版本 {#version-field}

版本，表示為 `major`。`minor`

類型: 字串
