# 開始使用內容卡 {#get-started-content-card}

>[!BEGINSHADEBOX]

**在此頁面上：**&#x200B;內容卡會直接在您的行動應用程式和網站中傳送個人化、非侵入式訊息，在內容中吸引使用者，而不會中斷他們原本要執行的操作。

>[!ENDSHADEBOX]

內容卡是 Adobe Journey Optimizer 新的數位傳訊功能，可直接在行動應用程式和網站提供個人化且吸引人的內容。 與傳統推播通知不同，內容卡可順暢整合到使用者介面，提供永久、非侵入式更新來增強使用者互動和體驗。

此功能可讓行銷人員向使用者呈現相關的豐富媒體內容，促進更高參與度，同時確保看到重要訊息，而不會中斷使用者歷程。

## 使用案例

當您想要讓訊息持續可供使用者依自己的步調採取行動，而不是在訊息到達時中斷訊息時，內容卡片會運作得最好。

| 優點 | 原因 | 範例使用案例 |
| --- | --- | --- |
| 持續可見度 | 卡片會保留在應用程式或網站中，直到使用者將其解除為止 | 進行中的促銷活動，功能強調使用者可以重新造訪 |
| 非侵入式傳遞 | 整合至介面，而非中斷使用者 | 被動更新會與其他內容一起顯示 |
| 無需推播選擇加入即可運作 | 傳遞的應用程式內或網站上，不受推播通知同意影響 | 觸及已停用推播通知的使用者 |
| 視覺效果豐富的訊息 | 支援媒體和call-to-action按鈕 | 影像的產品聚光燈和「立即購買」按鈕 |

## 何時不使用

內容卡片是供使用者以自己的步調檢視內容，因此不適用於每個案例。 在下列情況下考慮另一個管道：

* 此訊息是有時效性的警報，因為使用者可能無法及時開啟應用程式或網站來檢視資訊卡
* 目標使用者為非作用中，因為內容卡只會顯示在應用程式或網站內，非作用中的使用者永遠不會看到內容
* 內容是敏感的，或需要安全處理，而持續可見的卡無法保證安全
* 訊息需要緊急call to action，例如快閃銷售，推送通知或電子郵件可促使您更快速地採取行動
