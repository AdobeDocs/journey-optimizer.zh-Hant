# 內建權限 {#ootb-permissions}

內建許可權代表可以指派給&#x200B;**[!UICONTROL 角色]**&#x200B;的各種許可權，以微調您對Journey Optimizer的使用者存取權。 高階許可權包含低階許可權，詳細資訊位於[此頁面](high-low-permissions.md)。
